  • Introduction au Java Script

    • Avant de commencer

      • Avant de commencer, précisons que le JavaScript est un langage de programmation script, orienté objet, qui est responsable des différents comportements de la page web. L'intérêt des scripts est sans doute leur manière d'être utilisés : en effet, ils ne sont pas obligatoirement exécutés au chargement de la page. Ils sont lancés lorsqu'un événement spécifique se produit.
        Rappelons en effet que le HTML sert à insérer le contenu, la structure; le CSS permet de mettre en page le tout; le JS quand à lui permet de créer des animations et de créer des interactions avec l’utilisateur.

    • Comment insérer du JavaScript

      • Il y a deux façons d’insérer du JavaScript :
        - en l’insérant à n’importe quel endroit d’une page en intégrant
        <script type=« text/javascript » >
        //Code JavaScript
        </script>
        - en créant et en appelant un fichier extérieur
        <script type="text/javascript"
        src="chemin_du_fichier"> </script>
        // Permet d’écrire un commentaire sur une ligne
        /* permet d’écrite un commentaire sur plusieurs lignes */

  • Les débuts

    • Les commentaires

      • Afin de commencer avec de bonnes bases, habituons-nous dès à présent à utiliser des commentaires afin de faciliter la compréhension du code. Il faut savoir l’utiliser à bon escient.
        // Permet d’écrire un commentaire sur une ligne
        /* permet aussi d’écrire un commentaire sur une ligne */
        /* ou sur plusieurs lignes */

    • Les variables

      • Pour déclarer une variable rien de plus simple :
        var nom;
        Pour affecter une valeur à une variable :
        var age;
        age = 18
        Il est aussi possible d’affecter immédiatement une valeur, c’est à dire initialiser la variable :
        var age = 18
        Pour une variable message, on utilise des guillemets afin de l’ordinateur puisse délimiter le début et la fin de notre texte :
        var bonjour = « Salut Jojo, comment vas-tu ? »
        Il y a plusieurs types de variables :
        String —> Chaines de caractères
        Int —> nombres
        Float —> Nombres décimaux
        Bool —> True or false
        Array —> Tableau

    • La concaténation

      • Nous avons la possibilité d’afficher le nombre de frères de soeurs de l’utilisateur en mettant bout à bout trois chaines de caractères :
        var poisson = 2; // on cree la variable pour pouvoir tester
        alert(« ‘Vous avez’ + poisson + ‘poissons »); // on affiche les chaines mises bout-à-bout

    • Demander une chaîne de caractère au visiteur

      • Nous pouvons également demander directement l’âge du visiteur en utilisant « prompt » :
        var age = prompt(« Combien de poissons avez-vous ?");
        // on demande le nombre de poissons
        alert("Vous avez " + poisson + "poissons");
        // on affiche la phrase

    • Les variables sur les nombres

      • Sachez qu’il est possible de convertir une chaine de caractères en nombre ! Et cela grâce aux types de variables que nous avons vu précédemment :
        parseInt(chaine_a_convertir) (convertir en un nombre entier)
        parseFloat(chaine_a_convertir) (convertir en un nombre décimal).
        JavaScript nous permet aussi de calculer !
        Voici comment réaliser des opérations :
        Les opérateurs arithmétiques : +, -, /, *,%
        L'opérateur de concaténation : + (pour les strings)
        Les opérateurs de comparaisons : <, >, ==, <=, >=, !=, ===
        Les opérateurs logiques : &&, ||
        Exemple :
        var i = 50;

        var z = i + j;
        document.write(z);

  • Les fonctions

    • Les fonctions sur les nombres

      • Quelques fonctions sur les nombres :
        Math.ceil : retourne l’entier supérieur ou égal au paramètre.
        Math.floor: retourne l’entier inférieur ou égal au paramètre.
        Math.min: retourne le plus petit de paramètres.
        Math.round: retourne l’entier arrondi le plus proche.
        Math.abs : retourne la valeur absolue.
        Math.random: retourne une valeur aléatoire compris entre 0 et 1 (pas de paramètre)
        Exemples :
        Math.abs(-5.6);
        Math.ceil(5.6);
        Math.floor(5.6);
        Math.floor(-10.3);
        Math.round(5.6);
        Math.round(5.3);
        Math.max(10, 5.1, 4);

        On fait suivre cette déclaration du contenu de la fonction (es instructions de celle-ci).
        S'il y a plusieurs instructions, on délimite le contenu de notre fonction grâce à des accolades { et } :
        on forme alors ce qu'on appelle un bloc d'instructions.
        length
        Calculer la longueur d’une chaîne : length
        var txtAccueil=« Bonjour";
        var longueurTxt=txtAccueil.length;
        charAt()
        Accès à un caractère d’une chaîne : charAt()
        var txtAccueil="Bonjour";
        var lettre=txtAccueil.charAt(3); //j

      • substring()
        Extraire d’une chaîne de caractère : substring()

        chaine.substring(3) -> du 4 ème caractère à la fin
        substr()

        chaine.substr(debut,nombre de caractère à inclure)

  • Les tableaux

    • Les tableaux

      • Un tableau permet de stocker des données. C’est idéal pour stocker les informations lorsqu’elles deviennent nombreuses.
        Un tableau s’enregistre dans une variable.
        Pour créer un tableau, on utilise new Array().
        Donc var jour = new Array();
        Pour initialiser un tableau on utilise les parenthèses après Array, en précisant dans l'ordre, et en les séparant par des virgules, les valeurs de notre tableau.
        var jours = new Array ("Lundi", "Mardi", "Mercredi", "Jeudi", "Vendredi", "Samedi", « Dimanche »);

    • Parcourir un tableau

      • Pour accéder à un élément, on utilise tableau[indice].
        Pour ajouter un nouvel élément, on modifie simplement sa valeur.
        var jours = new Array ("Lundi", "Mardi", "Mercredi", "Jeudi", "Vendredi", "Samedi"); alert("La première case vaut : " + table[1]);
        // on lit l'element d'indice 1 qui est Mardi table[2] = "Meduse"; // on le modifie
        table[6] = "Dimanche"; // on crée un nouvel élément

  • Le DOM et sa manipulation

    • Le DOM

      • On parle souvent du DOM (Document Object Model) en javascript mais plus souvent de sa manipulation. En effet la manipulation du DOM permet de changer le contenu ou/et l’affichage de la page HTML en fonction des actions de l’utilisateur. Cela se passe en plusieurs étapes.
        Dans un premier temps il faut récupérer le noeud html sur lequel on veut agir. Pour cela il existe plusieurs commandes en javascript.
        document.getElementById(“monIdentifiant”) qui permet de récupérer un élément avec son ID, ici l’id : monIdentifiant.

        document.querySelector(“#monIdentifiant”) qui permet de récupérer le premier élément ayant la class ou identifiant donnée, ici l’identifiant : monIdentifiant. ATTENTION avec le .querySelector, il faut respecter la syntaxe CSS. Ainsi ne pas oublier le “.” pour les class et le “#” pour les identifiants.

        document.querySelectorAll(“.mesClass”) qui permet de récupérer l’ensemble des éléments ayant la class donnée, ici la class : mesClass. ATTENTION le résultat de la selection de l’ensemble des éléments aura pour résultat un tableau javascript. Ainsi il faudra dans la plupart des cas utiliser une boucle et une incrémentation pour appliquer nos transformations à chaque élément ayant cette même classe.

    • Manipulation css

      • Une fois nos éléments sélectionnés généralement dans une variable prédéfinie, par exemple : var annonceModulaire = document.querySelector(“#welcomeTitle”), nous allons voir quelles propriétés nous allons pouvoir lui attribuer :
        Ajouter/Changer une propriété CSS à un élément : pour changer une propriété css, nous allons utiliser le .style auquel nous rajouterons la propriété sur laquelle nous souhaitons agir par exemple sa couleur avec le .color. Une fois la propriété sélectionnée, pour rentrer un paramètre il faut respecter la syntaxe suivante
        noeud.style.propriete =”parametre”; Ainsi ici pour changer la couleur, en rouge par exemple, de mon élément précédemment annoncé j’écrirais ceci :
        annonceModulaire.style.color=”red”;
        ATTENTION les propriétés contenant des “-” doivent être remplacées par leur équivalent en camelCase, par exemple la propriété : background-color devient backgroundColor.
        N’indiquer aucun paramètre à la proprieté a pour but de retourner le parametre/valeur de cette propriété. Par exemple :
        console.log(annonceModulaire.style.color)
        retournera la couleur de l’élément de la variable annonceModulaire à savoir l’élément ayant l’identifiant #welcomeTitle.

    • Manipulation HTML

      • Pour la plupart des actions nous utiliserons le .innerHTML
        En effet pour retourner le contenu HTML d’un élément, dans la console par exemple, procéderons ainsi
        console.log(annonceModulaire.innerHTML);
        Pour modifier le contenu annonceModulaire.innerHTML=”Nouveau contenu”;
        ATTENTION si vous souhaitez inscrire des balises de code HTML veillez à utiliser au sein de vos “” des ‘’ afin d'éviter toute fermeture non voulue du paramètre. Par exemple
        annonceModulaire.innerHTML=” <p class=”description”>Notre description </p>”;
        annonceModulaire.innerHTML=” <p class=’description’>Notre description </p>”;

    • Remarques


      • L’utilisation du DOM est généralement effectuée dans des fonctions déclenchées par des événements (exemple : au clic, au survol).

        Privilégier la stylisation de vos éléments par votre page CSS au possible
